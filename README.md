# Flask Login and Registration System
A simple user authentication system built with Flask and MySQL that allows users to register, log in, and maintain sessions through a web interface.

## Features
- User Registration
- Secure Login Authentication
- Session Management
- Logout Functionality

## Technologies Used
- Python
- Flask
- MySQL
- HTML
- CSS

## How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Ria-yad/flask-login-registration.git
```

2. Install dependencies

```bash
pip install Flask flask-mysqldb
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```
http://127.0.0.1:5000
```
